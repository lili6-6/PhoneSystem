# PhoneSystem
开发时间：2026.1-2026.3
简介：3d音乐节奏类游戏的手机系统，用于推进游戏剧情，以及包含货币系统，场景切换系统，任务系统
说明：在这个系统中，进一步规范了代码架构，遵守Manager+Controller的结构，从上到下层层管理，实现数据与UI分离。
 设计开发了多个app，聊天、日历+事件簿、记账、相机、相册、地图。
 其中：
 聊天app实现通过Mnagaer里的数据动态生成联系人，支持多联系人对话，分别保存聊天记录到本地。支持发送相册图片。接入阿里云百炼ai智能体，实现联系人信息回复，可通过提前设置的剧 情走向，来干预ai回复。
 日历app实现游戏内单独一套时间系统，通过scriptObject可配置事务，将会在对应的日期下显示待办事务。
 记账app实现游戏内的货币系统，可详细记录每一笔交易的事件用途备注对象等信息，并且支持实时的交易对象筛选。
 相机app实现手机camera的画面实时显示到屏幕上，支持前后双镜头，拍照后存储至本地。
 相册app实现相机拍的照片的展示，支持删除操作，同时也能在聊天界面中进行选择发送。
 地图app实现游戏中切换场景操作。
分工说明：手机系统为单独开发，已预留美术资产接入口，
